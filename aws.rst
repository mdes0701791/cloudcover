Amazon Web Services (AWS)
=========================
Amazon is the defacto standard for cloud computing resources.
For learning purposes you can (read: Should) create an account for free that can be tied into your standard Amazon account you use for purchases. You will be able to stand up and utilize the most basic (read: smallest) EC2 Instance for free with limited bandwidth and compute capacity. For learning purposes this is very adequate however it will only scratch the surface of the 100+ services AWS provides.
